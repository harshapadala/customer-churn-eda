About the Project

This project focuses on performing Exploratory Data Analysis (EDA) on the House Price dataset to understand the data patterns, missing values, feature relationships, and factors affecting house prices.

The analysis helps in preparing the dataset for future machine learning models.

Project Goals

Load and understand the dataset

Perform basic exploratory data analysis

Detect and visualize missing values

Study distribution of important features

Analyze correlation with house prices

Prepare data for modeling

Technologies Used
